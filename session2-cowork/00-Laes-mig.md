# Velkommen til Session 2 - Claude Cowork

Denne mappe indeholder synesisk materiale fra det fiktive pensionsselskab **NordPension**. Intet er ægte. Tal, navne og hændelser er opdigtet udelukkende til workshop-brug.

## Hvad ligger i mappen?

| Fil | Format | Indhold |
|---|---|---|
| `00-Laes-mig.md` | Markdown | Denne fil |
| `Q1-2026-driftsrapport.docx` | Word | Driftsrapport for Member Services, Q1 2026 |
| `Bestyrelsesmoede-2025Q4.pptx` | PowerPoint | Sidste kvartals bestyrelsesdeck |
| `medlemstilfredshed-2025.csv` | CSV | Kvartalsvise NPS-tal pr. region (2025) |
| `klagestatistik-2025.csv` | CSV | Antal klager pr. kategori pr. kvartal (2025) |
| `Forecast-2026.xlsx` | Excel | Indtaegter, omkostninger og nogletal for 2026 |
| `MTU-2025-resultater.pdf` | PDF | Medarbejdertrivselsundersogelse 2025 |

## Sadan bruger du mappen

I Session 2 monterer du mappen i Claude Cowork. Claude kan derefter laese, opsummere og krydsreferere indholdet med data fra NordPension-MCP-connectoren.

God arbejdslyst.
